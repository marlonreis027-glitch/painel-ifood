⚙ Configuração Supabase
1
Acesse supabase.com, crie conta e clique em New Project.
2
No menu esquerdo clique em SQL Editor, cole o SQL abaixo e clique em Run.
CREATE TABLE IF NOT EXISTS licencas ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, nome TEXT NOT NULL, email TEXT NOT NULL, chave TEXT UNIQUE NOT NULL, plano TEXT DEFAULT 'mensal', ativo BOOLEAN DEFAULT true, criado TIMESTAMP WITH TIME ZONE DEFAULT NOW() ); ALTER TABLE licencas ENABLE ROW LEVEL SECURITY; CREATE POLICY "leitura_publica" ON licencas FOR SELECT USING (true); CREATE POLICY "admin_tudo" ON licencas FOR ALL USING (true);
3
Vá em Settings → API Keys, copie a Project URL, a Publishable key (anon) e a Secret key (service role).
4
Cole as chaves no campo Editar credenciais abaixo e clique em Salvar.
